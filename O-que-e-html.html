What is HTML?

extensão .html é usada para identificar arquivos de HTML, que é a linguagem de marcação de hipertexto. O HTML é a base para a criação de páginas da web, permitindo a estruturação de conteúdos como textos, imagens, links e botões.

características do HTML

É uma linguagem de marcação de texto, não de programação

Define a estrutura e o significado do conteúdo da web

É fundamental para a criação de páginas de aplicações

É o bloco de construção mais básico da web

É usado para criar portais e documentos eletrônicos on-line

criou o HTML?

O HTML foi inventado por Tim Berners-Lee, um físico do centro de pesquisas CERN, na Suíça

A primeira versão foi publicada em 1991

O HTML5 foi lançado em 2014, introduzindo novas tags semânticas

Como abrir um arquivo HTML?

No Google Chrome no Windows, basta clicar com o botão direito do mouse no arquivo HTML e selecionar "Abrir com" e depois escolher o Google Chrome
