What is CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilo que define a aparência de páginas web. É usada em conjunto com HTML para criar sites.

O que o CSS faz?

Define a fonte, cor, espaçamento, margens e disposição dos elementos de uma página

Permite criar tabelas, ajustar imagens, usar variações de layouts

Permite reutilizar estilos e aplicar regras em cascata

Permite criar sites responsivos
